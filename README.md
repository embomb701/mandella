# The Heritage Gazette

A fake-but-straight-faced humor website styled like a real cultural news outlet, focused on Mandela Effect stories and collective-memory oddities.

## Concept

This site plays the premise completely seriously: it looks like an archival publication while covering famously misremembered pop-culture details like the Monopoly monocle, the Fruit of the Loom cornucopia, and the Sinbad genie movie that never existed.
